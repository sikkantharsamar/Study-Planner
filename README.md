# Study Planner Web App

Smart Study Planner - Web-based task manager for university students with task tracking, study sessions, progress visualization, and auto-notifications. Built with Next.js, TypeScript, and Tailwind CSS. Demonstrates modern web development with React hooks, context API, and responsive design.

**✨ KEY FEATURES**

🔐 User Registration & Authentication

📝 Task Management (Priority-based: HIGH/MEDIUM/LOW)

📅 Schedule Organization by time periods

⏱️ Study Session Tracking per subject

📊 Progress Visualization with progress bars

🔔 Intelligent Auto-Notifications

📈 Comprehensive Reports (Task & Study statistics)

## Getting Started

1. Install Node.js (https://nodejs.org)
2. Clone or navigate to the project directory
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Open http://localhost:3000 in your browser

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Context API for state management
